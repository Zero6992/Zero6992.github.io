ChatGPT Discord Bot 中文教學

Posted on Mar 10, 2023

此頁面為 https://github.com/Zero6992/chatGPT-discord-bot 的中文說明

功能

  • /chat 與ChatGPT聊天！
  • /private 切換至私人模式
  • /public 切換至公開模式
  • /replyall 在回覆所有模式和預設模式之間切換
  • /reset 清除ChatGPT的對話歷史
  • /chat-model 切換不同的聊天模型
    • 修改.env文件中的CHAT_MODEL可更改預設模型
    • OFFICIAL: GPT-3.5模型
    • UNOFFICIAL: ChatGPT Website model

聊天

/chat

模式

  • /pubalic（預設模式） 機器人直接在頻道上回覆
  • /private 機器人的回覆只能被使用該命令的人看到
  • /replyall 機器人將回覆頻道中的所有訊息，而不使用斜槓命令（/chat也將無法使用）

     警告 在/raplyall模式下，機器人很容易被觸發，可能會導致程式失敗

設置

安裝前的必要設置

  • 推薦的Python版本為3.10
  • 在安裝好的資料夾下執行pip3 install -r requirements.txt
  • 將文件的名稱.env.dev更改為.env

步驟1：創建Discord機器人

  1. 前往 discord developers 創建一個 Application

  2. 在Application下建立Discord機器人

  3. 從機器人設置中獲取token

  4. 將token存儲到.env中的DISCORD_BOT_TOKEN下

  5. 將「MESSAGE CONTENT INTENT」選項打開

  6. 通過OAuth2 URL生成器邀請網址將你的機器人邀請加入你的伺服器

步驟2：官方API身份驗證

生成OpenAI API金鑰

  1. 前往https://beta.openai.com/account/api-keys

  2. 點擊「創建新的秘密金鑰」

  3. 將秘密金鑰存儲到.env中的OPENAI_API_KEY下

  4. 你現在可以進行步驟3了

步驟2：ChatGPT模型網站身份驗證-2種方法

電子郵件/密碼身份驗證（不支持Google/Microsoft帳戶）

  1. 在 openAI 上創建帳戶

  2. 將您的電子郵件保存到.env的OPENAI_EMAIL下

  3. 將您的密碼保存到.env的OPENAI_PASSWORD下

  4. 你現在可以進行步驟3了

Session Token 身份驗證

  1. 前往https://chat.openai.com/chat 登入

  2. 使用F12打開控制台

  3. 打開Application(應用程式) > Cookies

  4. 從Cookies中複製__Secure-next-auth.session-token的值，並將其粘貼到.env中的SESSION_TOKEN下

  5. 你現在可以進行步驟3了

步驟3：在你的電腦運行機器人

  1. 打開終端機或命令提示字元

  2. 到你安裝ChatGPT Discord機器人的目錄下

  3. 執行python3 main.py以啟動機器人

步驟3：使用Docker運行機器人

  1. 構建Docker映像和運行Docker容器docker compose up -d

  2. 檢查機器人是否運作正常docker logs -t chatgpt-discord-bot

    停止機器人：

    • docker ps 查看正在運行的服務列表
    • docker stop <BOT CONTAINER ID> 停止運行中的機器人

祝你聊天愉快！

額外設定：禁用logging記錄

  • 在.env中將LOGGING的值設為False

額外設定：設置機器人啟動收到的訊息

  • 機器人第一次啟動或重置時會調用啟動訊息
  • 可以通過修改starting-prompt.txt中的內容來設置
  • 文件中的所有文字都將作為訊息傳送給機器人
  • 在你的Discord頻道中獲取ChatGPT Bot的第一條消息

    1.  右鍵單擊要接收消息的頻道，複製ID

    2.  將其粘貼到.env中的DISCORD_CHANNEL_ID下
